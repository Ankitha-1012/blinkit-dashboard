# blinkit-dashboard
📊 Blinkit Sales Dashboard (Power BI)
🔍 Overview

This project presents an interactive Power BI dashboard built using Blinkit sales data.
The dashboard provides meaningful insights into sales performance, product categories, and customer trends to support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance
Identify top-performing product categories
Understand customer purchasing behavior
Track order and revenue trends over time
🛠️ Tools & Technologies
Power BI
Microsoft Excel (for data preprocessing)
📈 Key Insights
Identified high-revenue generating product categories
Observed sales trends across different time periods
Highlighted patterns in customer orders and demand
Detected variations in effort vs. sales performance
📊 Dashboard Features
Interactive filters and slicers
Category-wise sales analysis
Time-based trend visualization
KPI indicators for quick insights
